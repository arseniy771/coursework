Глава 3. Функциональные возможности графического редактора Gimp

В настоящее время наблюдается растущий интерес к графическим редакторам. Программа GIMP, предназначенная для работы с растровыми изображениями, представляет собой мощный инструмент для создания и редактирования изображений.

GIMP способен эффективно конкурировать с Adobe Photoshop, хотя пока не получил широкого распространения в коммерческих проектах. Несомненным преимуществом GIMP является его бесплатное распространение. Функциональность программы высока для удовлетворен потребностей даже опытных и требовательных пользователей, работающих с растровой графикой.

GIMP - пакет для создания и редактирования растровых изображений (растровый графический редактор), разрабатываемый сообществом разработчиков по технологии разработки с открытым исходным кодом (Open Source) и распространяемый на условиях свободной лицензии GNU (GNU Public License — GPL). GIMP прекрасно подходит для любительской и полупрофессиональной работы с изображениями: обработки фотографий, создания графических композиций и коллажей, создания элементов дизайна web-страниц.

Программное обеспечение GIMP предоставляет пользователям возможность полноценной работы с растровой графикой без необходимости приобретения дорогостоящих коммерческих пакетов или использования их нелегальных копий. Это особенно важно в условиях ужесточения законодательства об авторском праве в России.

Разработка проекта GIMP началась в 1995 году в Калифорнийском университете в Беркли. Открытая модель разработки и актуальность задачи создания мощного бесплатного инструмента для работы с растровой графикой способствовали активному развитию проекта, которое продолжается по сей день. Вначале для графического интерфейса GIMP использовалась библиотека Motif, но затем была создана новая библиотека GTK (GIMP ToolKit), ставшая одной из ведущих свободных библиотек для графических интерфейсов (в настоящее время используется GTK2+).

Ключевые особенности GIMP:

* Свободная модель распространения: проект развивался на протяжении более десяти лет и продолжает своё развитие, а последние версии доступны всем.

* Кроссплатформенность: GIMP работает на всех дистрибутивах Linux, всех ветках BSD-систем, MacOS и MS Windows.

* Гибкость и расширяемость: GIMP имеет встроенный язык для создания плагинов, а также возможность расширения функциональности с помощью языка Python.

Последняя стабильная версия GIMP – это GIMP 2.6.10. В настоящий момент данный графический редактор может считаться достойным конкурентом Adobe Photoshop в плане функциональных возможностей по созданию и редактированию изображений, и это несмотря на его бесплатный характер.

Интерфейс GIMP может показаться непривычным для рядового пользователя, поскольку рабочая область изображения, инструменты и палитры располагаются в отдельных окнах. Такая структура была обусловлена первоначальной ориентацией GIMP на работу с несколькими рабочими столами в среде GNU/Linux, где мультитаскинг является стандартным.

Несмотря на это, GIMP обладает рядом неоспоримых преимуществ: во-первых, он является постоянно развивающимся проектом (более 10 лет разработки и перспективы дальнейшего развития), во-вторых, он кроссплатформенен (доступен для всех дистрибутивов Linux, всех ветвей BSD, MacOS и MS Windows), в-третьих, он отличается гибкостью и расширяемостью, поскольку оснащен встроенным языком для создания расширений и API для интеграции с Python.

GIMP содержит 2 основных окна – это окно инструментов (Рис 2.) или Toolbox и окно изображения — Image Window (Рис 3.).

Кроме этих окон можно открыть (из меню, вызываемого через команду Файл - Диалоги) различные вспомогательные окна, служащие для выбора инструментов, шаблонов, задания цвета (палитра), слоев (layers) и т. д.

Панель инструментов GIMP представляет собой центральный и неотъемлемый компонент приложения, который не может быть удален или скрыт. В ней собраны основные инструменты для работы с графикой.

Для удобства пользователей предусмотрены подсказки: при наведении курсора мыши на любой элемент панели инструментов в течение короткого промежутка времени появляется всплывающее окно с описанием его функции. Кроме того, нажатие клавиши F1 при выделенном элементе панели инструментов вызовет контекстную справку.

Меню панели инструментов – особое. Оно содержит некоторые команды, которые не найти в прикрепляемых к изображению меню. Здесь включены команды для настроек, создания определенных типов диалогов, и т.д.

Пиктограммы инструментов. Пиктограммы являются кнопками, которые активируют инструменты для разнообразных действий: выделение частей изображений, рисования, преобразования, и т.п. Раздел Введение в панель инструментов описывает принципы работы с инструментами. Каждый инструмент описан в разделе Инструменты.

Цвета фона/переднего плана. Область выбора цвета показывает текущий выбранный цвет переднего плана и фона, который применяется во многих операциях. Щелчок по одному из них вызовет палитру цветов. Щелчок по двунаправленной стрелке поменяет местами два цвета, щелчок по небольшому символу в нижнем левом углу сбросит их в черный и белый цвета.

Кисть/Текстура/Градиент. Расположенные здесь значки показывают текущие выбранные элементы GIMP-а, такие как:

— кисть, используемую всеми инструментами, позволяющими рисовать по изображению ("рисование" включает в себя такие операции как стирание и размазывание);

— текстуру, используемую для заливки выделенных областей изображения;

— градиент, который применяется когда необходимо создать плавный цветовой переход.

Щелчок по любому из этих значков вызовет диалоговое окно позволяющее изменить данный элемент.

Активное изображение. В GIMP можно работать с многими изображениями одновременно, но в любой момент только одно из них является "активным изображением". Здесь присутствует кнопка представления уменьшённой копии активного изображения в виде пиктограммы. Щелчок по ней вызовет диалог со списком всех открытых на данный момент изображений, позволяя при желании изменить активное изображение. Просмотр "активного изображения" по умолчанию отключен.

Программа GIMP, как и любое другое программное обеспечение, используемое регулярно, требует индивидуальной настройки под задачи конкретного пользователя.

Несмотря на то, что в процессе установки программы многие параметры настраиваются автоматически, а остальные задаются по умолчанию и подходят для большинства случаев, ознакомление с возможностями настройки GIMP может оказаться полезным. Пользователь всегда может захотеть или понадобиться изменить стандартные настройки.

Среди дополнительных возможностей редактирования GIMP предоставляют различные экранные фильтры, такие как:

* симуляция различных видов дальтонизма;

* коррекция гаммы;

* коррекция контраста;

* управление цветом.

Диалог настройки вызывается из меню главного окна (называемого также «Панель инструментов») командами «Файл / Настроить...» (рис. 4).

Первый пункт настроек— настройка окружения (ресурсов программы). Можно оставить все эти настройки в исходном состоянии (по умолчанию), поскольку они вполне разумны.

Индивидуальную настройку имеет смысл начать с пункта «Дисплей» (рис. 5).

Здесь можно определить внешний вид «клеточек» для прозрачного фона изображения, однако изменения параметров этих «клеточек» в большинстве случаев не нужны. А вот разрешение монитора - очень важный параметр, влияющий на многие другие свойства программы. Современные видеосистемы и версии пакета GIMP позволяют правильно определить разрешение в автоматическом режиме, однако для особых случаев есть возможность установить разрешение вручную, получив значения либо от видеосистемы (например, командой xdpyinf о в Linux), либо в результате калибровки, измеряя в миллиметрах длины показываемых на экране эталонных линеек.

На вкладке «Интерфейс» (рис. 6) можно определить размеры окон предпросмотра слоев и каналов и размеры окна навигации. Окно навигации (рис. 6) - это вспомогательное окно, обеспечивающее быстрое изменение масштаба изображения в окне изображения. «Ползунок» в нижней части окна навигации обеспечивает непрерывное изменение масштаба, а кнопки «+» и « - » - ступенчатое. В итоге в окне изображения будет показана область, ограниченная рамочкой в окне навигации.

Важная часть интерфейса программы — клавиши быстрого доступа («горячие клавиши»). Именно в пункте настроек интерфейса имеется возможность их включить, выключить и переопределить.

Переопределение существующих и определение новых клавиш быстрого доступа осуществляется с помощью диалога «Настроить клавиши быстрого доступа» (рис. 8), в котором для каждого элемента каждого пункта меню программы показаны существующие «горячие клавиши». Определение «горячих клавиш» делается очень просто нужно выделить пункт меню (действие) и нажать требуемую комбинацию клавиш.

После настроек интерфейса имеет смысл перейти к параметрам сетки (рис. 9). Если предполагается использовать сетку для облегчения точного позиционирования элементов изображения, то настройки по умолчанию требуют изменения, например, в соответствии с рис. 8 видно, что пересчёт точек экрана в единицы длины производится в соответствии с установленным разрешением экрана.

Теперь перейдём к настройкам параметров инструментов (рис. 10). Здесь также может потребоваться изменение настроек по умолчанию.

Во-первых, полезно включить режим «Сохранять параметры инструментов при выходе», чтобы каждый раз не устанавливать всё заново.

Во-вторых, если предполагается использовать сетку для облегчения точного позиционирования элементов изображения, то значение параметра «Дистанция прилипания» полезно сделать меньшим, чем половина шага сетки. Так, для сетки с шагом в 10 точек этот параметр имеет смысл сделать равным 3 или 4 точкам.

Также может быть полезно установить параметр «Делать слой или контур активным» для инструмента перемещения, чтобы сократить количество переключений между окнами программы. Эффект от наличия или отсутствия этого режима легко обнаружить опытным путём.

В настройках панели инструментов (Главного окна программы) можно сделать одно-единственное изменение — включить режим показа используемых кистей, текстур и градиентов (рис. 11).

При изменении параметров видеосистемы (например, после смены видеокарты или замены монитора) в настройках изображения по умолчанию (рис. 12) нужно проделать скучную, но необходимую работу для всех вариантов шаблонов изображений, включая собственный вариант, установить правильные значения разрешения экрана. При необходимости разорвать связь между разрешением по горизонтали и по вертикали нужно щелкнуть мышью на соответствующем значке в «Расширенных параметрах».

Наконец, в настройках окна изображения (рис. 13) можно изменить вид инструмента рисования. Удобным вариантом представляется пиктограмма инструмента с перекрестьем. Пиктограмма позволяет видеть, какой инструмент используется в данный момент, а перекрестье позволяет точно увидеть точку изображения, с которой ведется работа. Что касается оставшихся настроек, то здесь вполне можно положиться на параметры по умолчанию.

Настройка управления цветом является отдельной сложной задачей. Используемые по умолчанию параметры устроят большинство пользователей, преимущественно публикующих фотографии в Сети. Все остальные настройки можно легко постигнуть на основе личного опыта.
